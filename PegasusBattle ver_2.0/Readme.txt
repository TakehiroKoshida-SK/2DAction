■ゲーム名■
ペガサスバトル

■説明■
超簡単操作の対戦型2Dアクションゲームです。
最大4人まで同時プレイ可能。
プレイヤーは6種のユニットから1体を選び、13×20マスのマップ場で戦闘を行います。
2人、または4人が戦い合うバトルロワイヤルモード、
2対2に分かれて敵陣地への到達を競うチームバトルモードも。

■動作環境■
WindowsXP 以降
DirectX9 以降がインストールされていること

■起動方法■
「PegasusBattle_window.exe」又は「PegasusBattle_fullscrean.exe」を起動してください。
それぞれウィンドウ版、フルスクリーン版になります。

■操作説明■
基本的な操作は同梱のマニュアルをご覧ください。
USBゲームパッドで遊ぶことを前提としています。
接続されたゲームパッドの数が足りない場合は、キーボードのキーに操作が割り当てられます。
割り当てられるキーはマニュアル後方に記載しています。
ただし、同時押しが正常に判定できない理由で一つのキーボードで複数のキャラを動かすことは現実的じゃないです。

また、どうも正常に動くゲームパッドは限られているようです。
現状動作確認が出来ているゲームパッドは「iBUFFALO CLASSIC USB GAMEPAD」です。

■インストール■
ゲームプログラムの圧縮フォルダを任意の場所に展開してください。

■アンインストール■
ゲームプログラムを展開したフォルダごと削除してください。

■注意事項■
このゲームの転載、使用素材の流用、データの抽出などを禁止します。
